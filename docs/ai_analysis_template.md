# AI 深度分析模板

## 题目详情页 AI 深度解析内容结构

### 1. 知识点分析
```
【知识点】
- 主要知识点：三角形、勾股定理
- 相关考点：直角三角形、斜边计算
```

### 2. 错因诊断
```
【错因分析】
你在计算斜边长度时，使用了勾股定理 a² + b² = c²，但在计算过程中出现了错误：

❌ 你的计算：3² + 4² = 13
✅ 正确计算：3² + 4² = 9 + 16 = 25

错误原因：在将3²和4²的值相加时出错。
```

### 3. 解题思路
```
【解题思路】
1. 识别题目类型：这是一个直角三角形求斜边的问题
2. 确定已知条件：两条直角边分别为3和4
3. 选择合适的方法：使用勾股定理
4. 正确计算：
   - 先计算各边的平方：3² = 9，4² = 16
   - 将平方数相加：9 + 16 = 25
   - 求平方根：√25 = 5
5. 得出答案：斜边长度为5
```

### 4. 知识拓展
```
【知识拓展】
勾股定理是几何学中的基本定理，适用于所有直角三角形。
在实际应用中，常用于：
- 计算距离
- 测量高度
- 建筑设计
```

### 5. 练习建议
```
【练习建议】
建议练习类似的勾股定理应用题，熟练掌握：
- 平方数的计算
- 平方根的求解
- 实际问题的转化
```

## 不同题型的分析模板

### 数学题模板

#### 计算题
```
【知识点】计算题 - 四则运算

【你的答案】3×4=13
【正确答案】3×4=12

【错因分析】
你在计算3×4时，错误地将结果算成了13。
✅ 乘法口诀：三四十二

【易错点提醒】
- 乘法口诀需要熟练记忆
- 可以通过分解验证：3×4 = 3+3+3+3 = 12
```

#### 应用题
```
【知识点】应用题 - 行程问题

【题目解析】
这是一个相遇问题，关键信息：
- 甲乙两地相距120千米
- 甲车速度40千米/时
- 乙车速度60千米/时

【解题步骤】
1. 计算相对速度：40 + 60 = 100 千米/时
2. 计算相遇时间：120 ÷ 100 = 1.2 小时 = 1小时12分钟

【你的错误】
在计算相对速度时，你使用了减法而不是加法。
```

#### 几何题
```
【知识点】几何题 - 三角形

【图形分析】
题目中的图形包含：
- 直角三角形ABC
- 直角在∠B处
- AB = 3，BC = 4

【解题思路】
根据勾股定理求斜边AC：
AC² = AB² + BC²
AC² = 3² + 4² = 25
AC = 5

【你的作答】
你正确地识别出了直角三角形，但计算时出现了错误。
```

### 英语题模板

```
【知识点】语法 - 时态

【题目】
She _____ to school every day.
A. go B. goes C. going

【你的选择】A
【正确答案】B

【解析】
这是一道一般现在时的题目。
关键信息："every day" 表示习惯性动作

第三人称单数的规则：
- 一般情况加 -s：go → goes
- 以s, x, ch, sh结尾加 -es：watch → watches
- 以辅音+y结尾，变y为i加 -es：study → studies

【记忆技巧】
第三人称单数变化，就像给动词穿上一件"s"外套
```

## 前端展示格式建议

### 格式化渲染
- 使用 Markdown 渲染支持数学公式
- 关键错误用 ❌ 标记
- 正确内容用 ✅ 标记
- 代码块用于展示计算过程

### 视觉层次
- 一级标题：【知识点】
- 二级标题：具体内容
- 列表用于步骤展示
- 代码块用于数学公式

### 交互元素
- 可折叠的详细解析
- 相关题目推荐
- 知识点链接跳转
