# TINT 小红书宣发完整方案

## 一、战略定位层

### 1.1 品牌人格化定位

**品牌名**：TINT（There Is No Teacher）

**核心slogan**：AI时代，每个孩子都能成为自己的老师

**人设定位**：用AI重新定义学习的「数据科学家家长」

**情绪价值**：把"焦虑"转化为"洞察"——从"不知道孩子学得怎么样"到"看透孩子学习数据的每一个细节"

---

### 1.2 用户心智锚定

| 传统认知 | TINT想要建立的认知 |
|---------|-------------------|
| 搜题工具=作弊 | TINT=学习数据资产化 |
| 批改=给个分数 | TINT=发现学习问题的显微镜 |
| 错题=失败 | TINT=学习证据+优化线索 |
| 老师负责 | TINT=AI时代的自主学习闭环 |

---

## 二、内容矩阵：五大支柱

### 支柱一：「家长焦虑共鸣」系列（引流爆款）

**核心逻辑**：先戳痛点 → 再给希望 → 最后给工具

#### 选题库

**标题公式1**：《为什么我[放弃传统方法]？因为[数据真相]太残酷了》
→ 《为什么我不再逼孩子刷题了？因为AI让我看到了真相》

**标题公式2**：《[数字]天，从[情绪崩溃]到[掌控感]》
→ 《7天，从"辅导作业到心梗"到"一眼看穿孩子哪里不会"》

**标题公式3**：《那些[竞争对手]的家长，最后都[来了这]》
→ 《那些放弃作业帮的家长，最后都在用这个"笨办法"》

#### 视频结构（45-60秒）

```
0-3s：情绪钩子
"辅导作业3年，我才发现我一直做错了…"

3-15s：痛点放大
"每天2小时，发脾气、吵架、孩子哭、我崩溃…"

15-35s：认知转折
"直到我意识到：AI时代，最贵的不是老师，是数据"

35-50s：产品演示
"现在，3分钟批改+自动归档+7天出报告"

50-60s：CTA
"首批内测开放中，留言【体验】获取邀请"
```

---

### 支柱二：「数据洞察震撼」系列（差异化展示）

**核心逻辑**：用"数据可视化"建立专业度和信任

#### 选题库

```
《我儿子7天的数学作业，AI告诉我这3件事…》
《同样考80分，为什么一个能逆袭，一个会崩盘？（附数据）》
《把一学期错题变成"学习诊断报告"，这个功能太吓人了》
《AI看完我家娃30天作业，说出了我永远发现不了的问题》
```

#### 内容形式
- 录屏演示：真实作业 → 批改过程 → 报告生成
- 数据对比：第1天 vs 第7天的可视化变化
- 逻辑拆解：为什么"3次以上才出报告"

#### 关键话术
> "单次作业有偶然性，但3次以上就能找到规律。这就是为什么TINT至少积累3天才出分析报告——我们要的不是分数，是真实的学习状况。"

## 附录：产品核心信息速查

### 产品定位
- **代号**：TINT（There Is No Teacher）
- **核心功能**：家庭作业批改、错题AI辅导、错题数据采集、错题分析报告
- **产品内核**：个人学习数据资产化

### 目标用户
- **核心用户**：中学生家长
- **用户痛点**：焦虑（担心帮不到孩子、看不到进步）
- **使用场景**：家长辅导学生作业

### 差异化优势
- 不只是搜题工具，而是数据驱动的学习系统
- 需要3次以上数据积累才出报告（排除偶发性，找共性）
- 苏格拉底式启发教学（不直接给答案）
- 数据纠偏机制（人工判断错题/对题）

### 产品亮点
- 拍照上传，2-3分钟完成批改
- 错题自动归档，带图题目有切片
- 数学、英语两科目支持
- 准确率高
- 成本：一页A4约2毛钱

### 商业模型
- 配额充值（类订阅，限制有效期）
- 有免费用户体验
- 首批用户特殊权益
